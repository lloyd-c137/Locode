# 更新日志

本文档记录 Locode 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-30

### 新增
- 🎉 首次发布 Locode
- 🤖 集成 AI 编程助手功能
- 📁 本地文件系统操作支持
- 💻 集成 Monaco Editor 代码编辑器
- 📝 AI 对话功能，支持自然语言交互
- 🔒 工作空间安全访问限制
- ⚡ AI 响应流式输出
- 🔧 灵活的 API 配置管理
- 📊 SQLite 数据库存储配置
- 🎨 暗色主题界面
- 📱 响应式布局，支持移动端
- 🏷️ 标签页系统（编辑器、预览、终端、设置）
- 🖥️ 终端集成，支持命令执行
- 🎮 代码预览功能
- ⚙️ 设置面板，支持多配置管理
- 🔄 实时文件刷新
- 📂 文件树浏览器
- 📄 文件创建、读取、修改、删除
- 🔍 文件搜索功能
- 🌐 支持多种 AI 模型 API
- 🎯 手动输入模型名称
- 🔐 API Key 安全存储
- 💾 配置导入/导出
- 📋 剪贴板操作支持
- ⌨️ 键盘快捷键
- 🖱️ 触摸设备优化

### 安全
- 文件访问限制在工作空间内
- API Key 服务器端存储
- 路径验证防止目录遍历攻击

### 技术栈
- Node.js + Express
- 原生 JavaScript + Monaco Editor
- SQLite 数据库
- Server-Sent Events (SSE)
